# E-Commerce Sales Analytics Dashboard

## Project Overview
Built a 3-page interactive e-commerce analytics 
dashboard using real customer dataset with 3,900 
records across 25 product categories and 50 locations.

## Tools Used
- Power BI — Dashboard and Visualization
- PostgreSQL — Star Schema Design
- Power Query — Data Cleaning and ETL

## Dashboard Pages

### Page 1 — Sales Performance
- Current Purchases grew 135.74% over previous period
- Total Revenue: $233.08K vs Previous $98.7K
- Clothing is top category  in sales
- Peak sales in November and December

### Page 2 — Customer Review Analytics
- 3,900 total customers across 50 locations
- Average rating: 3.75/5
- 78.28% positive reviews (3,053 customers)
- Montana has highest reviews (96)
- Customer satisfaction rate: 78.25%

### Page 3 — Customer Purchase Insights
- Total purchase value: $331.95K
- Average purchase amount: $85.12
- Fall season drives highest purchases ($84.36K)
- Clothing top category with 1,737 customers
- Most common purchase frequency: Every 3 Months

## Key Business Insights
1. Purchase growth of 130.83% shows strong momentum
2. Clothing dominates all categories in revenue
3. Fall is highest revenue season at $84.36K
4. 78.28% customer satisfaction shows healthy retention
5. Average customer age 44 — mature audience targeting
